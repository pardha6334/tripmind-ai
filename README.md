# TripMind AI ✈️
> Plan smart. Travel safe.

## Problem Statement
Travel Planning & Experience Engine - Plan trips dynamically with preferences, constraints, and real-time updates.

## Solution
AI-powered travel planner that detects risks BEFORE planning, gives user control, and produces complete day-wise itineraries with live data.

## Google Services Used
| Service | Purpose |
|---------|---------|
| Gemini 2.5 Flash API | Core AI engine |
| Google Search Grounding | Live web data |
| Firebase Hosting | GCP deployment |
| Firebase Analytics | Usage tracking |
| Google Fonts | Typography |

## Architecture
4-screen intelligent flow:
1. Smart Input Form
2. Live Risk Scanner
3. Conflict Resolution
4. Full Itinerary Output

## Accessibility

## Security
- Zero hardcoded API keys
